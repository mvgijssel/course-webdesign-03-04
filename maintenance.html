The directory structure

All the different pages are in the root directory. The images are placed in a folder called 'images', the javascript used is placed in the folder 'javascript', the stylesheets in the folder 'stylesheets' and the design elements used in the folder 'design'.

The stylesheets used

There are two stylesheets 'styles.css' and 'alt_styles.css'. The first is for the normal layout and the second is used for the visually impaired.

The images (dimensions, compression and transparency)

The logo, everybody loves Reactjs, is a PNG with transparency and dimensions 498x161. The other images used are JPEG, with quality 80 (20% compression). The dimensions of the image on the keynote page is 400x267. The images used on the committee page are all 200x200.

The fonts that are used

The font used for the normal text is the standard sans-serif font. The font for all the headlines is the standard serif font. The font used in the logo is Helvetica Neue.

The search engine optimization techniques that were used

The following SEO techniques were used:

  • Added a robots.txt file to the root of the website
  • Used alternative text on images
  • The filenames of the images represent the content of the images themselves
  • Added a meta description tag the the home page
  • The titles of all the pages reflect their content
